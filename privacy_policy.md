## Overview
Tabs Exporter is a browser extension that allows users to export their open browser windows and tabs into a local file.

## Data Collection
This extension does **not** collect, store, transmit, or share any personal or browsing data.

## Data Usage
All data processing happens locally in the user's browser and is initiated explicitly by the user.

## Third Parties
Tabs Exporter does not use third-party services, analytics, tracking tools, or external APIs.

## Permissions
- **Tabs & Windows**: Used to read open tabs and windows for export file locally purposes.
- **Downloads**: Used to save the generated file locally.

## Contact
If you have any questions about this privacy policy, you can contact the developer via the GitHub repository.
